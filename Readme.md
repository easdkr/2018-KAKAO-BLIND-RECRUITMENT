### 1. 비밀 지도

##### 문제 링크 : https://programmers.co.kr/learn/courses/30/lessons/17681

#### 구현 설명
 ```
 OR 연산을 사용할 수 있는지 묻는 문제이다.
 주의할 점 : | 연산 결과를 #과 ' '의 문자열로 만들 때, 
 문자열의 길이가 주어진 arr의 길이보다 작을 경우 문자열의 앞에 ' '를 추가해야한다
 ex) 4 = 000100(2) = '()()()#()()' (arr의 길이가 6인 경우)
 ```
